## Bidra til nettsiden?

### Hva du trenger

Før du begynner, sørg for at du har følgende installert:
- [Node.js](https://nodejs.org/) (versjon 16 eller nyere)
- npm (følger med Node.js)

### Installasjon

1. **Klon repositoriet**
   ```bash
   git clone https://github.com/TIHLDE/drift.git
   cd drift
   ```

2. **Installer avhengigheter**
   ```bash
   npm install
   ```

3. **Start utviklingsserveren**
   ```bash
   npm run dev
   ```

   Nettsiden vil nå være tilgjengelig på `http://localhost:5173/`


### Arbeidsflyt

Når du skal gjøre endringer i prosjektet:

1. **Opprett en ny branch**
   ```bash
   git checkout -b feature/bidrag
   ```

2. **Gjør dine endringer** og commit dem
   ```bash
   git add .
   git commit -m "Beskrivelse av endringen"
   ```

3. **Push branchen til GitHub**
   ```bash
   git push origin feature/bidrag
   ```

4. **Opprett en Pull Request** på GitHub som må godkjennes av nettsideansvarlig før den merges inn i hovedbranchen.
